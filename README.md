# Bluestock Mutual Fund Analytics

## Project Overview
End-to-end Mutual Fund Analytics Platform built as part of Bluestock Fintech Data Analyst Internship.

## Dataset
- 40 mutual fund schemes across 10 fund houses
- 46,000 NAV records from 2022-2026
- 32,778 investor transactions

## Deliverables
- ✅ D1 — ETL Pipeline
- ✅ D2 — SQLite Database
- ✅ D3 — EDA Analysis
- ✅ D4 — Performance Metrics (CAGR, Sharpe, Beta, VaR)
- ✅ D5 — Streamlit Dashboard
- ✅ D6 — Advanced Analytics & Fund Recommender
- ✅ D7 — Final Report & Presentation

## Bonus
- ✅ B2 — Streamlit web app as Power BI alternative

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, SQLite, Streamlit, ReportLab, python-pptx

## How to Run
```bash
pip install -r requirements.txt
python scripts/etl_pipeline.py
streamlit run scripts/dashboard.py
```
